# StarParks Customer Service Project 🎮🌏

## 项目概述

StarParks游戏应用的Discord客服系统，专为东南亚多语言环境设计。

## 核心功能

- ✅ Discord论坛自动管理与回复
- ✅ 东南亚多语言支持（印尼语、泰语、越南语、马来语、英语）
- ✅ 游戏行业专项翻译
- ✅ 跨文化沟通适配
- ✅ 客服质检与优化

## 技术栈

- **平台**: Discord Forum Channels
- **AI助手**: OpenClaw
- **语言支持**: Indonesian, Thai, Vietnamese, Malay, English, Chinese
- **技能系统**: SkillHub技能生态
